📊 Attendance Management System

Smart solution for managing student attendance

📌 Project Description

This web-based system helps automate attendance management in colleges. It allows admins, faculty, and students to manage and track attendance efficiently.

🎯 Features

  • 👨‍🎓 Student Dashboard
  • 👩‍🏫 Faculty Attendance System
  • 🏫 Admin Panel
  • 📅 Subject-wise Attendance
  • 📈 Reports & Analytics
  • 🔐 Secure Login System

🛠️ Technologies Used

React.js
Node.js
Express.js
MongoDB

⚙️ Installation

Clone the repository and run using npm:
